# 🤖 AI FAQ Chatbot

AI FAQ Chatbot is an intelligent web-based question answering system built using **React, Node.js, Express, and NLP (Natural Language Processing)**.  
It matches user queries with the most relevant FAQ using similarity matching and returns instant answers in a chat-style interface.

---

## 🚀 Features

- AI-powered FAQ matching system  
- Natural Language Processing (NLP) based question detection  
- Smart similarity matching (Jaro-Winkler algorithm)  
- Real-time chat interface  
- Clean modern UI design  
- User & Bot message bubbles  
- Enter key support for sending messages  
- Clear chat functionality  
- Fully responsive (Mobile + Desktop)  
- Splash screen with AI branding  

---

## 🛠️ Technologies Used

**Frontend:**
- React (Vite)
- Tailwind CSS
- Axios

**Backend:**
- Node.js
- Express.js

**NLP / AI Logic:**
- Natural npm library
- Jaro-Winkler similarity algorithm

---

💡 How It Works
User types a question in the chat
Backend receives the message
NLP compares input with stored FAQs
Best matching question is found
Corresponding answer is returned
Response is displayed in chat UI
